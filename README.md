# -Pok-dex-Replica
Projet interactif reproduisant un Pokédex : l’utilisateur peut saisir le nom d’un Pokémon et obtenir sa fiche caractéristiques complète. Cette application m’a permis de pratiquer la manipulation du DOM, les requêtes de données et la gestion d’interactions utilisateur en JavaScript.  ⚙️ Technologies : HTML • CSS • JavaScript
